# Contributing to Trend Engine

We welcome open-source contributions. 

## Workflow
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add some amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Areas for Contribution
- **Data Integrations**: Connecting to X/Twitter APIs, Reddit, or RSS feeds natively.
- **Knowledge Graphs**: Adding more advanced entity extraction or timeline modeling to Cytoscape.
- **Language**: Expanding to French, Spanish, Russian, or Chinese.

Please ensure your code passes ESLint rules and matches the `tailwnd css` bento-grid styling patterns used throughout the application.
